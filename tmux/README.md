# tmux の構成ファイル
- tmux の構成ファイルは`.tmux.conf`です．
- 配置場所は `~/.tmux.conf` です．

## Usage
```bash
cd
curl -O https://raw.githubusercontent.com/solareenlo/dotfiles/master/tmux/.tmux.conf
```
